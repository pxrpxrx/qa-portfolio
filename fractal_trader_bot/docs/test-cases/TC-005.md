# ТЕСТ-КЕЙС #005

---

## 1. Заголовок
Подготовка ордера по сигналу LONG

---

## 2. ID
TC-005

## 3. Приоритет
P0 (Blocker)

---

## 4. Предусловия
- BingXOrderPreparer инициализирован
- Сигнал LONG (UP) с symbol="BTC-USDT", price=50000, trend="UP"
- В сигнале присутствуют stop_loss и take_profit

---

## 5. Шаги

| # | Шаг | Ожидаемый результат |
|:---|:---|:---|
| 1 | Вызвать `prepare_signals([signal_long])` | Возвращается список из 1 prepared-ордера |
| 2 | Проверить `direction` | "UP" |
| 3 | Проверить `entry_order.side` | "BUY" |
| 4 | Проверить `entry_order.positionSide` | "LONG" |
| 5 | Проверить `entry_order.type` | "MARKET" |
| 6 | Проверить наличие `stop_loss_order` | Присутствует, type="STOP_MARKET", side="SELL" |
| 7 | Проверить наличие `take_profit_order` | Присутствует, type="TAKE_PROFIT_MARKET", side="SELL" |
| 8 | Проверить `summary.entry_price` | 50000 |
| 9 | Проверить `summary.risk_reward_ratio` | 2.5 |

---

## 6. Ожидаемый результат
Ордер подготовлен корректно: entry (MARKET BUY LONG), стоп-лосс (STOP_MARKET SELL LONG), тейк-профит (TAKE_PROFIT_MARKET SELL LONG).
