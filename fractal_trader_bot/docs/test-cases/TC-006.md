# ТЕСТ-КЕЙС #006

---

## 1. Заголовок
Проверка минимальных стопов

---

## 2. ID
TC-006

## 3. Приоритет
P2 (Medium)

---

## 4. Предусловия
- positionSizing импортирован
- Используются минимальные значения ATR и цены, при которых стоп может быть < 0.2%

---

## 5. Шаги

| # | Шаг | Ожидаемый результат |
|:---|:---|:---|
| 1 | Вызвать `positionSizing(price=50000, atr=1.0, direction="UP", stop_loss_mult=1.25)` | ATR очень мал, стоп в процентах будет < 0.2% без защиты |
| 2 | Проверить `stop_percent` | Не менее 0.2% |
| 3 | Проверить `take_percent` | Не менее 0.6% |
| 4 | Проверить `stop_loss` | Меньше entry (для UP) |
| 5 | Сравнить с расчётом без защиты | Стоп должен быть выше минимального порога |

---

## 6. Ожидаемый результат
Даже при очень малом ATR стоп-лосс не опускается ниже 0.2%, а тейк-профит ниже 0.6%.
