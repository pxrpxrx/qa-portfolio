# ТЕСТ-КЕЙС #001

---

## 1. Заголовок
Расчет размера позиции через positionSizing

---

## 2. ID
TC-001

## 3. Приоритет
P0 (Blocker)

---

## 4. Предусловия
- Модуль positionSizing доступен для импорта
- Функция вызывается с корректными параметрами

---

## 5. Шаги

| # | Шаг | Ожидаемый результат |
|:---|:---|:---|
| 1 | Вызвать `positionSizing(capital=5000, price=50000, atr=200, direction="UP", risk_percent=0.02)` | Возвращается словарь с ключами `size`, `entry`, `stop_loss`, `take_profit` |
| 2 | Проверить `entry` | Равно 50000.0 |
| 3 | Проверить `stop_loss` | Меньше entry (для UP) |
| 4 | Проверить `take_profit` | Больше entry (для UP) |
| 5 | Проверить `risk_reward_ratio` | Больше 0 |
| 6 | Проверить `stop_percent` | Не менее 0.2% |
| 7 | Проверить `take_percent` | Не менее 0.6% |

---

## 6. Ожидаемый результат
Размер позиции рассчитан корректно, стоп и тейк уровни адекватны направлению, соблюдены минимальные проценты.
