# TC-002: Инициализация платежа без userId

**Приоритет:** P1 (Высокий)  
**Тип:** Негативный / Валидация  
**Метод:** GET  
**Endpoint:** `/api/sbp/init-and-qr`

## Предусловия

- Микросервис запущен и доступен

## Шаги

1. Отправить GET-запрос без параметра userId:
   ```
   GET /api/sbp/init-and-qr?amount=100.50
   ```

## Ожидаемый результат

- **Статус:** 400 Bad Request
- Тело ответа содержит описание ошибки валидации

## Постусловия

- Заказ не создан
- Платеж в Tinkoff не инициализирован
