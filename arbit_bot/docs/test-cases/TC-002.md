# ТЕСТ-КЕЙС #002

---

## 1. Заголовок
Ввод невалидного значения спреда

---

## 2. ID
TC-002

## 3. Приоритет
P1 (High)

---

## 4. Предусловия
- Пользователь зарегистрирован в боте
- Пользователь в меню настройки спреда

---

## 5. Шаги

| # | Шаг | Ожидаемый результат |
|:---|:---|:---|
| 1 | Нажать кнопку "Спред 📊" | Бот запрашивает значение |
| 2 | Ввести "абв" (не число) | Бот выводит ошибку: "Введите число" |
| 3 | Ввести "-5" (отрицательное) | Бот выводит ошибку: "Спред должен быть > 0" |
| 4 | Ввести "1000" (слишком большое) | Бот выводит ошибку: "Спред не может превышать 50%" |

---

## 6. Ожидаемый результат
Все невалидные значения отклоняются с понятной ошибкой
