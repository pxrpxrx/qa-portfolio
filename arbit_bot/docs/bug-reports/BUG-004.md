# БАГ-РЕПОРТ #004

---

## 1. Заголовок
Telegram-бот не обрабатывает спецсимволы при вводе названия монеты

---

## 2. Метка приоритета
**Приоритет:** P3 (Low)  
**Серьезность:** Trivial

---

## 3. Описание
При ручном вводе названия монеты с символом '/' (например, BTC/USDT) бот сохраняет это значение как есть, но при запросе к API бирж символ '/' приводит к ошибке 400 (Bad Request).

---

## 4. Шаги для воспроизведения
1. Нажать кнопку "Ввести монету вручную"
2. Ввести "BTC/USDT"
3. Отправить

---

## 5. Фактический результат
API биржи возвращает ошибку 400

---

## 6. Ожидаемый результат
Бот должен автоматически заменять '/' на '_' или предупреждать пользователя

---

## 7. Окружение
- Telegram Desktop 4.14.4
- n8n 0.238.0

---

## 8. Предполагаемая причина
Отсутствие валидации ввода пользователя
