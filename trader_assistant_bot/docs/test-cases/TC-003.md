# ТЕСТ-КЕЙС #003

---

## 1. Заголовок
Срабатывание стоп-лосса при убытке

---

## 2. ID
TC-003

## 3. Приоритет
P1 (High)

---

## 4. Предусловия
- BxMonitor инициализирован с замокированным trader
- В монитор добавлена LONG-позиция BTC-USDT с stop_percent=2.5%
- trader.get_positions() возвращает тестовые данные
- trader.close_position() успешно закрывает позицию

---

## 5. Шаги

| # | Шаг | Ожидаемый результат |
|:---|:---|:---|
| 1 | Вызвать _check_exit_conditions("BTC-USDT", pnl_percent=-3.0) | Позиция удаляется из монитора |
| 2 | Проверить monitor.positions | "BTC-USDT" отсутствует в словаре |
| 3 | Проверить для SHORT позиции: side="SELL", pnl=-3.0, stop=2.0% | Позиция удаляется из монитора |
| 4 | Проверить, что при pnl=1.0% (в пределах стопа) | Позиция НЕ удаляется |

---

## 6. Ожидаемый результат
При достижении стоп-лосса (PnL <= -stop_percent) позиция удаляется из монитора. При PnL в пределах нормы позиция остаётся.
