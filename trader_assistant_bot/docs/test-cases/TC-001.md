# ТЕСТ-КЕЙС #001

## 1. Заголовок
Расчет ATR на тестовых данных

---

## 2. ID
TC-001

## 3. Приоритет
P0 (Blocker)

---

## 4. Предусловия
- Модуль ATR (bx_atr.py) доступен
- Тестовые kline-данные подготовлены (16 свечей с high/low/close)
- API BingX замокирован (не требуется реальное подключение)

---

## 5. Шаги

| # | Шаг | Ожидаемый результат |
|:---|:---|:---|
| 1 | Передать 16 тестовых свечей в ATR.calculate() | Функция возвращает словарь с ключами `atr`, `atr_3`, `atr_percent` |
| 2 | Проверить значение ATR | ATR > 0 (положительное число) |
| 3 | Проверить ATR×3 | `atr_3 = round(atr * 3, 8)` |
| 4 | Проверить ATR% | `atr_percent` рассчитан относительно последней close-цены |
| 5 | Передать < 15 свечей | ATR.calculate() возвращает None |

---

## 6. Ожидаемый результат
ATR корректно рассчитывается на тестовых данных. При недостаточном количестве свечей возвращается None.
