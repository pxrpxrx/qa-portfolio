# БАГ-РЕПОРТ #001

---

## 1. Заголовок
ConfigLoader падает при отсутствии config.json

---

## 2. Метка приоритета
**Приоритет:** P1 (High)  
**Серьезность:** Critical

---

## 3. Описание
При запуске любого модуля, импортирующего `config_loader.py`, происходит падение с `FileNotFoundError`, если файл `config.json` не существует в директории `src/utilities/config/`. ConfigLoader в конструкторе открывает файл без проверки существования и без graceful fallback, что делает невозможным запуск бота без предварительного создания конфигурационного файла.

---

## 4. Шаги для воспроизведения
1. Склонировать репозиторий
2. Убедиться, что `src/utilities/config/config.json` отсутствует
3. Выполнить `python src/utilities/config_loader.py`
4. ИЛИ выполнить `python src/bx_trail_traider.py`

---

## 5. Фактический результат
```
FileNotFoundError: [Errno 2] No such file or directory: '.../src/utilities/config/config.json'
```
Крайне неинформативная ошибка без указания, какой файл нужно создать.

---

## 6. Ожидаемый результат
При отсутствии config.json ConfigLoader должен:
- Создать файл с шаблонными значениями по умолчанию,
- ИЛИ загрузить значения из переменных окружения,
- ИЛИ выдать понятное сообщение с инструкцией по созданию файла.

Никакой модуль не должен падать с `FileNotFoundError` при импорте.

---

## 7. Окружение
- Python 3.10+
- Windows 11 / Ubuntu 22.04
- Файл: `src/utilities/config_loader.py:8`

---

## 8. Предполагаемая причина
В `config_loader.py` строка 8:
```python
with open(config_path, 'r', encoding='utf-8') as f:
```
Отсутствует проверка `config_path.exists()`, блок `try/except` или `fallback` к значениям по умолчанию.

Также нет шаблонного файла `config.json` в репозитории, который можно скопировать.
