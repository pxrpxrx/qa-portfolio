# Регрессионный чек-лист: Криптообменник

---

## ✅ Клиентский бот

| # | Проверка | Статус |
|:---|:---|:---|
| 1 | Команда /start открывает главное меню | |
| 2 | Кнопка "Создать обмен" показывает выбор валюты | |
| 3 | Выбор BTC → запрос суммы в RUB | |
| 4 | Выбор RUB → запрос суммы в BTC | |
| 5 | Ввод суммы → запрос реквизитов карты | |
| 6 | Ввод реквизитов → подтверждение данных | |
| 7 | Подтверждение → заявка отправлена оператору | |
| 8 | Кнопка "Курсы и лимиты" показывает курсы | |
| 9 | Кнопка "Мои заявки" показывает историю | |
| 10 | Загрузка чека → статус "pending_verification" | |
| 11 | Чек пересылается в канал оператора | |
