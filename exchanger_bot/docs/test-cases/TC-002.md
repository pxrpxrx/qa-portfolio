# ТЕСТ-КЕЙС #002

---

## 1. Заголовок
Админ вводит реквизиты для заявки

---

## 2. ID
TC-002

## 3. Приоритет
P0 (Blocker)

---

## 4. Предусловия
- Есть заявка со статусом "pending_operator"
- Админ открыл админ-панель (/admin)

---

## 5. Шаги

| # | Шаг | Ожидаемый результат |
|:---|:---|:---|
| 1 | Нажать кнопку "🏦 Реквизиты на подтверждение" | Показывается список заявок |
| 2 | Нажать "▶️" для навигации | Переход к следующей заявке |
| 3 | Нажать "✅ Выбрать" | Появляются кнопки "Подтвердить" и "Отменить" |
| 4 | Нажать "✅ Подтвердить" | Бот запрашивает реквизиты |
| 5 | Ввести номер карты | Бот запрашивает банк |
| 6 | Ввести банк | Бот запрашивает ФИО |
| 7 | Ввести ФИО | Бот запрашивает сумму |
| 8 | Ввести сумму | Бот показывает подтверждение |
| 9 | Нажать "✅ Отправить клиенту" | Реквизиты сохранены, клиент получил их |

---

## 6. Ожидаемый результат
Реквизиты сохранены в БД, статус заявки — "awaiting_payment"
