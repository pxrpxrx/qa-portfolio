# БАГ-РЕПОРТ #003

---

## 1. Заголовок
Курс BTC/RUB отображается как 0.00 при недоступности BestChange API

---

## 2. Приоритет
P1 (High)

## 3. Серьёзность
Major

---

## 4. Описание
При недоступности BestChange API бот показывает курс 0.00, вместо того чтобы вывести сообщение об ошибке.

---

## 5. Шаги для воспроизведения
1. Отключить интернет или заблокировать запросы к BestChange
2. Нажать "Курсы и лимиты"

---

## 6. Фактический результат
Показывается "1 BTC = 0,00 RUB"

---

## 7. Ожидаемый результат
Сообщение "❌ Не удалось получить курсы. Попробуйте позже."

---

## 8. Рекомендация
В `get_btc_rub()` возвращать None при ошибке, а в `show_rates()` проверять `if btc_rub is None`
