## Вопросы для собеседования

**Q: Что такое Pairwise и когда его применять?**
A: Pairwise (попарное тестирование) — техника, при которой тестируются все пары значений параметров. Применяется когда 3+ параметра и полное перебрание невозможно (729 комбинаций → 12 тестов).

**Q: Какой инструмент для Pairwise ты использовал?**
A: PICT от Microsoft — генерирует минимальный набор тестов, покрывающих все пары. Альтернатива — AllPairs (онлайн).

**Q: Как тестировать responsive?**
A: Chrome DevTools → Toggle Device Toolbar (F12). Чеклист: header, контент, формы, footer на 3 разрешениях (1920, 768, 375).
