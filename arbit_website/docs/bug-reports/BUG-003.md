# БАГ-РЕПОРТ #003

---

## 1. Заголовок
Стартовая модалка закрывается только по крестику, не закрывается по клику на фон

---

## 2. Приоритет
P3 (Low)

## 3. Серьёзность
Minor

---

## 4. Описание
При открытии сайта стартовая модалка с предупреждением появляется автоматически. Закрыть её можно только нажав на крестик. Клик по фону (за пределами модального окна) не закрывает её.

---

## 5. Шаги для воспроизведения
1. Открыть сайт
2. Дождаться появления стартовой модалки
3. Кликнуть на тёмный фон за пределами модального окна

---

## 6. Фактический результат
Модалка не закрывается

---

## 7. Ожидаемый результат
Модалка должна закрываться при клике на фон

---

## 8. Предполагаемая причина
В обработчике событий стартовая модалка не подключена к общему обработчику закрытия по фону

---

## 9. Рекомендация
Добавить в JS-обработчик для стартовой модалки:

```javascript
startModal.addEventListener("click", (e) => {
    if (e.target === startModal) startModal.classList.remove("active");
});
