# БАГ-РЕПОРТ #002

---

## 1. Заголовок
В тёмной теме у изображений партнёров остаётся белый фон

---

## 2. Приоритет
P3 (Low)

## 3. Серьёзность
Trivial

---

## 4. Описание
При включении тёмной темы у логотипов партнёров в секции "Наши партнеры" остаётся белый фон, что выглядит неэстетично.

---

## 5. Шаги для воспроизведения
1. Открыть сайт
2. Нажать кнопку "🌙 Dark mode"
3. Прокрутить до секции "Наши партнеры"

---

## 6. Фактический результат
Логотипы отображаются на белом фоне

---

## 7. Ожидаемый результат
Фон должен быть тёмным или прозрачным

---

## 8. Предполагаемая причина
В CSS для `body.dark-mode .partners` задан белый фон (`background: #fff !important;`)

---

## 9. Рекомендация
Изменить CSS на:
```css
body.dark-mode .partners {
    background: transparent !important;
}
