# БАГ-РЕПОРТ #001

---

## 1. Заголовок
В тёмной теме текст в FAQ плохо читаем из-за недостаточного контраста

---

## 2. Приоритет
P2 (Medium)

## 3. Серьёзность
Minor

---

## 4. Описание
При включении тёмной темы текст ответов в FAQ становится плохо читаемым из-за низкого контраста между фоном и текстом.

---

## 5. Шаги для воспроизведения
1. Открыть сайт
2. Нажать кнопку "🌙 Dark mode"
3. Прокрутить до секции FAQ
4. Нажать на любой вопрос

---

## 6. Фактический результат
Текст ответа сливается с фоном, плохо читается

---

## 7. Ожидаемый результат
Текст должен быть контрастным и хорошо читаемым на тёмном фоне

---

## 8. Окружение
- Браузер: Chrome 114+
- Устройство: ПК / Мобильное

---

## 9. Предполагаемая причина
В CSS для `body.dark-mode .faq-item p` не задан явный цвет текста

---

## 10. Рекомендация
Добавить в CSS:
```css
body.dark-mode .faq-item p {
    color: #e0e0e0 !important;
}
