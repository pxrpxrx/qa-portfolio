# Тест-план: Сайт ARB (лендинг)

**Проект:** Landing Page для арбитражного бота  
**Версия:** 1.0  
**Дата:** 2025-06-21  
**Автор:** Константин Горбунов (QA Engineer)

---

## 1. Цель тестирования

Проверить корректность работы одностраничного сайта, включая:
- Отображение всех элементов
- Работу навигации и якорей
- Функциональность модальных окон
- Работу тёмной темы
- Адаптивность на мобильных устройствах
- Корректность ссылок на Telegram

---

## 2. Объекты тестирования

| Модуль | Описание |
|:---|:---|
| **Хедер** | Логотип, навигация, кнопка "Телеграм бот" |
| **Hero-блок** | Заголовок, изображение |
| **Функции** | 4 карточки с описанием |
| **Интерфейс** | Изображение + список настроек |
| **Сигналы** | Изображение + список описаний |
| **Тарифы** | 3 карточки с ценами, кнопка "Выбрать тариф" |
| **Партнёры** | 5 логотипов бирж с анимацией |
| **FAQ** | 4 вопроса с раскрывающимися ответами |
| **Контакты** | Кнопка перехода в Telegram-канал |
| **Футер** | Навигация, кнопка "Публичная оферта" |
| **Модальные окна** | Стартовая, оферта, согласие на обработку данных |
| **Тёмная тема** | Переключение через кнопку |
| **Адаптивность** | Отображение на мобильных устройствах |

---

## 3. Типы тестирования

| Тип | Описание |
|:---|:---|
| **Функциональное** | Проверка работы всех интерактивных элементов |
| **UI/UX** | Проверка внешнего вида, читаемости, удобства |
| **Кроссбраузерное** | Проверка в Chrome, Firefox, Safari |
| **Адаптивность** | Проверка на мобильных устройствах (320px, 375px, 768px) |
| **Регрессионное** | Проверка после изменений |

---

## 4. Инструменты тестирования

| Инструмент | Назначение |
|:---|:---|
| **Chrome DevTools** | Проверка адаптивности, отладка CSS |
| **Lighthouse** | Оценка производительности и доступности |
| **BrowserStack** | Кроссбраузерное тестирование (опционально) |
| **Вручную** | Проверка UI/UX, навигации, модалок |

---

## 5. Критерии качества

- Все ссылки ведут на корректные страницы/якоря
- Модальные окна открываются и закрываются корректно
- Тёмная тема применяется ко всем элементам
- Нет критических багов (P0)
- Сайт корректно отображается на мобильных устройствах

---

## 6. Риски

| Риск | Митигация |
|:---|:---|
| Некорректное отображение тёмной темы | Проверить все элементы вручную |
| Проблемы с адаптивностью | Использовать Chrome DevTools для эмуляции устройств |
| Ссылки на бота не работают | Проверить все ссылки вручную |
