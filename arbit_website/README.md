# ARB — Landing Page для арбитражного бота

**Одностраничный сайт (лендинг)** для продвижения Telegram-бота по мониторингу межбиржевых спредов.

---

## 📌 Описание проекта

Сайт предназначен для ознакомления пользователей с функционалом бота, его тарифами и преимуществами.  
Содержит информацию о биржах-партнёрах, FAQ, тарифах и ссылки на Telegram-бота.

---

## 🛠️ Технологии

| Компонент | Технология |
|:---|:---|
| **Вёрстка** | HTML5, CSS3 (без фреймворков) |
| **Анимации** | CSS transitions, hover-эффекты |
| **Модальные окна** | Нативный JavaScript |
| **Тёмная тема** | CSS-класс `dark-mode` |
| **Адаптивность** | Медиа-запросы (mobile-first) |

---

## 📂 Структура сайта

| Раздел | Описание |
|:---|:---|
| **Хедер** | Логотип, навигация, кнопка перехода в бота |
| **Hero** | Заголовок и главное изображение |
| **Функции** | 4 карточки с описанием возможностей бота |
| **Интерфейс** | Изображение + список настроек бота |
| **Сигналы** | Изображение + описание работы сигналов |
| **Тарифы** | 3 тарифа с ценами и кнопкой выбора |
| **Партнёры** | Логотипы бирж (Bybit, BingX, MEXC, HTX, Gate.io) |
| **FAQ** | 4 вопроса с раскрывающимися ответами |
| **Контакты** | Ссылка на Telegram-канал |
| **Футер** | Навигация, ссылка на оферту (модалка) |

---

## 🧩 Модальные окна

| Модалка | Триггер | Назначение |
|:---|:---|:---|
| **Стартовая** | Открывается автоматически | Предупреждение об инвестиционных рисках |
| **Согласие на обработку данных** | Кнопка из стартовой модалки | Согласие пользователя на обработку персональных данных |
| **Публичная оферта** | Кнопка в футере | Юридический документ с условиями использования |
