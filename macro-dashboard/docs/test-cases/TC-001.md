# TC-001: Расчёт процентного изменения (pctChange)

## Описание
Проверка корректности расчёта процентного изменения между текущим и предыдущим значением.

## Предусловия
- Список значений содержит минимум 2 элемента
- Значения являются валидными числами (не NaN)

## Шаги воспроизведения
1. Создать список значений: `[100.0, 110.0]`
2. Вызвать `DataManager.pctChange(values, 1)`
3. Проверить результат

## Ожидаемый результат
- Возвращаемое значение: `0.10` (10% рост)
- Точность: `±0.001`

## Постусловия
- Функция не модифицирует исходный список

## Приоритет
**P0** — критическая функция для расчёта momentum

## Связанные тесты
- `DataManagerTest.pctChange.positiveChange`
- `DataManagerTest.pctChange.negativeChange`
- `DataManagerTest.pctChange.nullList`
