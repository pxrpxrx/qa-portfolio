# TC-002: Оценка макро-среды (MacroContext)

## Описание
Проверка корректности оценки макро-среды (GREEN/YELLOW/RED) на основе 4 факторов.

## Предусловия
- Snapshot содержит валидные данные: M2, DGS10, DGS3MO, DXY, BAA10Y
- История M2 содержит минимум 5 элементов

## Шаги воспроизведения
1. Создать Snapshot с бычьими сигналами:
   - M2 растёт (history: `[20.0, 20.5, 21.0, 21.5, 22.0]`)
   - Term spread нормальный (DGS10 > DGS3MO)
   - DXY слабеет (history: `[105.0, 104.5, 104.0, 103.5, 103.0]`)
   - HYG spread низкий (BAA10Y < 2.0)
2. Вызвать `new MacroContext(snap).assess()`
3. Проверить статус

## Ожидаемый результат
- `status()` = `"GREEN"`
- `m2Momentum()` > 0
- `termSpread()` > 0
- `explanation()` содержит описание факторов

## Постусловия
- MacroResult immutable

## Приоритет
**P0** — ключевая функция для принятия торговых решений
